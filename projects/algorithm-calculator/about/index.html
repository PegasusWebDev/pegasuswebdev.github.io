Project info:

expand_more What it does

The Algorithm Calculator

is a web application that is given a type of problem and two numbers, and solves the problem, showing its work on the screen.


The calculator goes through the problem step-by-step and uses the standard algorithm to solve the problem, just like a person would.

expand_more The coding

This project is

100% my code.

No libraries were used.


This program took me a few months to make, with the rest of the project in between.

The program uses a client-side web language called JavaScript, along with HTML and CSS.


You can see the code here.

expand_more How to use the app

Click "Try the Application" at the top of this page to proceed to the application page.

On the app, select an expression type from the dropdown menu. Currently supported are addition, subtraction, and multiplication.

Upon selecting an algorithm type, text boxes will appear, where you can type in the numbers for your expression.

Click "Calculate" to display the algorithm on the screen. You can then press "Next" to go through the steps.
